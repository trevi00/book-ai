int : 정수
float : 부동 소숫점
str : 문자열
bool : 부울


문자열
문자열.count("궁금한단어")
>>궁금한단어가 얼마나 있는지 카운트한 결과값
len(문자열)
>>문자열에 문자가 얼마나 있는가
문자열.find("궁금한단어")
>>궁금한단어가 어디에 있는지 숫자로 알려줌
문자열.replace("바꿀단어", "바꾸고싶은단어")
>>바꿀단어가 바꾸고싶은단어로 바뀜


문자열 출력
%s - string 문자열
%c - character 한글자
%d - int 정수
%f - float 부동 소숫점 ; f앞에 정수를 쓰면 소숫점 몇자리까지 표현할 건지 설정가능


포맷

f스트링
print(f"{변수}")

일반적
print("{}".format("넣을거"))
{}에 넣을거가 넣어짐
ex)
print("{0} {1}".format("데이터1", "데이터2"))
{0}은 데이터1을 지칭하고
{1}은 데이터2를 지칭함

변수
변수 = 0.887
print(format(변수, ".2f")
>>두자리수까지만 출력함
