1. Class Inheritance (상속)
  - **추상화(abstraction)**: 
  여러 클래스에 중복되는 속성, 메서드를 하나의 기본 클래스로 작성하는 작업
  - **상속(inheritance)**: 
  기본 클래스의 공통 기능을 물려받고, 다른 부분만 추가 또는 변경하는 것
    + 이 때 기본 클래스는 
    부모 클래스(또는 상위 클래스), Parent, Super, Base class 라고 부름
    + 기본 클래스 기능을 물려받는 클래스는 
    자식 클래스(또는 하위 클래스), Child, Sub, Derived class 라고 부름

  - 코드 재사용이 가능, 공통 기능의 경우 기본 클래스 코드만 수정하면 된다는 장점
  - 부모 클래스가 둘 이상인 경우는 다중 상속 이라고 부름
