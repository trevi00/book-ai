문자열에 있는 특정 문자 갯수 세기 (count 함수)
count('찾을 거')    # 문자열에 D 가 몇 번 나올까요? 대소문자도 구별함

문자열에 있는 특정 문자의 위치 알려주기 (index 함수)
index('찾을 거')   # 맨 앞 자리부터 0, 1, ... 순으로 위치를 표시

find 함수는 해당 문자가 없으면 -1을 리턴함 ( 에러는 안냄 )
find('찾을 거 ')    # 맨 앞 자리부터 0, 1, ... 순으로 위치를 표시

문자열 사이에 다른 문자 넣기 (join 함수)
join(string)     # 껴넣을 문자.join(문자열)

문자열 앞뒤에 공백 지우기 (strip 함수)
strip()      # 앞 뒤 공백이 여러개 있어도 한번에 다 지움
lstrip()      # 앞쪽(문자열 왼쪽)만 지우기 left strip 의 약자
rstrip()      # 앞쪽(문자열 오른쪽)만 지우기 right strip 의 약자

소문자를 대문자로 바꾸기 (upper)
upper()        # 본래 대문자는 놔두고, 소문자인 문자만 대문자로 변경

대문자를 소문자로 바꾸기
lower()        # 본래 소문자는 놔두고, 대문자인 문자만 대문자로 변경

문자열을 나누기
split()       # split() 인자를 넣지 않으면 디폴트로 스페이스를 기준으로 분리

문자열 중 일부를 다른 문자로 바꾸거나, 삭제하기
replace("A", "B")     # A 를 B 로 바꾸기
